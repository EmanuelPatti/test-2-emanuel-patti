Le principali carrateristiche proprie del CSS

Il CSS è un linguaggio di stile che dona alla nostra pagina una proprià identità cercando di renderla originale, CSS sta per Cascading Style Sheets che sarebbe il principio a cascata ovvero che i comandi vengono eseguiti dall'alto verso il basso. Inoltre possiamo pure utilizzare degli identificatori o classi che richiamati sul foglio di stile potranno sovrascrivere determinati dati messi da noi in precedenza essendo loro più 'forti'.

Le principali differenze tra JS e Typescript


la principale forma di polimorfismo che c'è in Javascript

Penso che la principale forma di polimorfismo che c'è in Javascript son gli Array perchè possono contenere qualsiasi tipo di dato al loro interno

    I principali concetti di programmazione di JavaScript includono:

  • 1. Variabili: le variabili consentono di assegnare un valore a un nome.
  • 2. Funzioni: le funzioni sono blocchi di codice che eseguono un'azione specifica.
  • 3. Oggetti: gli oggetti consentono di raggruppare dati e funzionalità in un unico elemento.
  • 4. Array: gli array sono strutture dati che consentono di contenere più valori in una singola variabile.
  • 5. Condizioni: le condizioni consentono di eseguire determinati blocchi di codice solo se si verifica una determinata condizione.
  • 6. Cicli: i cicli consentono di eseguire ripetutamente un blocco di codice finché non viene soddisfatta una determinata condizione.
  • 7. Eventi: gli eventi consentono di eseguire un'azione quando si verifica determinato evento, come il clic del mouse su un elemento HTML.
